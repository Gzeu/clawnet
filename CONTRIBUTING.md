# Contributing to ClawNet

🎉 Mulțumim pentru interesul tău în contribuirea la **ClawNet**! 

## 📌 Cum poți contribui
1. **Raportează bug-uri** sau **propune feature-uri** prin [GitHub Issues](https://github.com/georgeclaw/clawnet/issues).
2. **Trimite PR-uri** pentru bugfix-uri sau îmbunătățiri.
3. **Îmbunătățește documentația** (README, docstrings, exemple).
4. **Scrie teste** pentru funcționalități noi/existente.

## 🚀 Proces de contribuire
1. **Fork** repository-ul și creează un branch nou:
   ```bash
   git checkout -b feat/my-new-feature
   ```

2. **Scrie cod** respectând:
   - [Conventional Commits](https://www.conventionalcommits.org/)
   - TypeScript best practices
   - Testare acolo unde este posibil

3. **Rulează testele locale:**
   ```bash
   pnpm test
   ```

4. **Trimite un PR** cu:
   - Descriere clară a modificărilor
   - Referință la issue-ul rezolvat (dacă există)
   - Capturi de ecran (pentru UI changes)

## 📜 Cod de conduită
- Fii respectuos cu ceilalți contributori.
- Comentează codul acolo unde este necesar.
- Menține un ton constructiv în discuții.

## 📄 Licență
