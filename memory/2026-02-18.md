### API Endpoints

- `GET /health` - Health check
- `POST /api/v1/agents/register` - Înregistrează agent
- `GET /api/v1/agents` - Listează agenți
- `POST /api/v1/agents/query` - Caută agenți
- `POST /api/v1/memory/write` - Scrie în memorie
- `GET /api/v1/memory/:key` - Citește din memorie
- `POST /api/v1/memory/search` - Caută în memorie
- `WS /ws` - WebSocket pentru real-time
